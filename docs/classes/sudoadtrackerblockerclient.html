Class SudoAdTrackerBlockerClient

This is the main class used for Ad/Tracker blocking. Each instance of SudoAdTrackerBlockerClient will contain a filtering engine that can be configured to use a set of blocking rulesets. To query the filtering engine, you can call (@link SudoAdTrackerBlockerClient.checkUrl}.
