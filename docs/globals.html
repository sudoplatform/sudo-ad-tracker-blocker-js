WasmInitInput

WasmInitInput: string | RequestInfo | URL | Response | BufferSource | Module | Promise<Response | BufferSource | Module>
  • Defined in wasm.ts:17

The return value of initWasm. This will indicate how to load a Web Assembly file.

In a browser context, the easiest option is to specify a string that describes the fetch location of the web assembly file, relative to the website's root, e.g. :/web-assembly-files/file.wasm.

In NodeJS it is easiest to load the file using fs.readFile and provide the BufferSource of the file.
